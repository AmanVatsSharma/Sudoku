## Sudoku Ultimatum v1.0.2

Android release build: offline 9×9 Sudoku with progression, achievements, and themes.

### Highlights

- **Five difficulties** — Easy through Ultimatum
- **Notes, undo**, and up to **three hints** per puzzle
- **XP, levels, ranks** (Novice → Ultimatum)
- **Achievements**, streaks, best times, solve history, **resume** in-progress games
- **Light / dark** themes and accent colors
- **Offline-first** — progress stays on the device

### Download

- **`Sudoku-Ultimatum-1.0.2-multiabi.apk`** (~58 MB; in this repo under `releases/`) — universal APK with **armeabi-v7a**, **arm64-v8a**, **x86**, and **x86_64** native libraries.

**SHA256** `531c86430878e799d9b4f3c32c37d45f56e81457c0fd99df789587b16582a5a9`

### Install

Download the APK, open it on your Android device, and allow installation from your file manager or browser if prompted. See the repository README [Download (Android)](https://github.com/AmanVatsSharma/Sudoku/blob/main/README.md#download-android) for more detail.

### Requirements

- Android **7.0+** (API 24) typical for this build; use your device’s install prompts as the source of truth.

### Privacy

No gameplay data is sent to external servers. Settings and progress remain on the device.
