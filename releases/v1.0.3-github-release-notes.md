## Sudoku Ultimatum v1.0.3

Android release: offline 9×9 Sudoku with Ultimatum-style solving tools, refreshed board UI, and existing progression features unchanged.

### Highlights — gameplay (Ultimatum / legacy parity, additive)

- **Hypothesis branches** — create, switch, merge, and delete branches; drawer with previews; per-branch contradiction detection; branch state saved on resume.
- **Candidate tools** — fill-all pencil marks from true candidates; keypad aligned with candidates; optional **block invalid notes** with warning (Settings).
- **Contradiction banner** when an empty cell has no legal candidates.
- **Flow state** — streak-based flow with countdown; UI ring/label; **+20% base XP** on win if still in flow; **In The Flow** achievement.
- **Brancher** achievement for using branches.
- **Number pad modes** (Settings) — **bottom bar** (default) or **floating** pad near the selected cell.
- **Controls** — Undo, Erase, Notes, Hint, Branch; fill-candidates; note warnings; flow timer when active.
- **Win screen** — shows flow XP bonus when it applied.
- **Home / settings** — updated chips and toggles for the above.
- **Persistence** — saves branches, flow fields, `blockBad`, and `numberPadMode` (older saves migrate safely).

### Highlights — board and visuals

- **Standard 3×3 blocks** — heavy borders every third row/column; thin lines between cells (hairline on iOS, 1px on Android).
- **Closed outer frame** on the board (right/bottom match major stroke).
- **Per-edge border colors** so majors stay dark without tinting minor lines.
- **Theme tokens** `gridMinor`, `gridMajor`, `gridBoard` (light + dark).
- **Card-style board** — rounded corners (~22px), shadow/elevation, hairline border, padding; paper-like cell base via `gridBoard`.
- **Selected cell** — soft accent glow (iOS shadow / Android elevation).
- **Digits** — slightly larger relative to cell, tighter letter spacing, tabular figures on iOS.
- **Android** — ripple feedback on cells.
- **Sizing** — cells up to **48px** (minimum **34px**) on larger phones; layout accounts for card padding.

### Still included from earlier releases

- **Five difficulties**, notes, undo, hints, **XP / levels / ranks**, achievements (including solve milestones), streaks, history, **resume**, light/dark themes, **offline-first** play, daily reminder settings.
